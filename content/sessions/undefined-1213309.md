AI Agents are moving from the cloud to edge devices. openvela is an open-source operating system built on Apache NuttX, deployed on over 100 million devices. This talk introduces the openvela lightweight native AI Agent engine — a pure C implementation requiring as little as 100KB RAM — enabling watches, speakers, glasses, earbuds, and other AI hardware to make autonomous decisions and deliver proactive services.Outline:
1. The Trend: AI Agents Moving from Cloud to Edge
2. openvela AI Agent Architecture Deep Dive
3. Real-World Scenario Demonstrations
4. Open Source Ecosystem and Future Outlook
