Icehouse（Iceberg + Trino）的一项基石需求就是数据摄入。其中一种方式是借助 Apache NiFi。NiFi 是一款多模态的数据流水线工具，拥有大量处理器（processor），可以组装成一条流水线来满足你的具体场景。NiFi 的低代码/无代码方式，让数据工程师能够快速构建、部署并监控自己的数据摄入与转换流水线。NiFi 还支持用多种语言（包括 Java 和 Python）开发自定义处理器。

本次演讲将依次介绍几种常见做法，并最终演示一条丰富的数据流水线：从 Kafka 取数，执行典型的转换处理（包括数据富化），然后把数据加载到一张高性能的 Iceberg 表中，再由 Trino 进行消费。
