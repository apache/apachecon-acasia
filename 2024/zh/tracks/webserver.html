Web Server/Tomcat

出品人 : 张乎兴

Web 服务器在互联网时代扮演了至关重要的作用，在多年的发展中依然历久弥坚，保持自己的活力，显示出其强大的生命力，这些项目包括但不限于 Apache Tomcat / Apache Http Server 为代表的 Web 服务器 ，以及 Apache Sling / Apache Traffic Server / Apache Traffic Control 等更多的生态项目。

在这里，您将了解 Web 服务器领域的最新成果，一线大厂在生产当中的最佳实践，以及这些项目的最新发展规划。
