基于 Apache Doris 的 AI4Data 决策平台：生产落地与实践

奕嘉 苏

中文演讲 2026-08-07 15:00 GMT+8  (ROOM : 静明厅) #dataai

作为 Apache Doris 2025 MVP、Doris-MCP 核心贡献者以及 Yiwen Data Agent 联合创始人，我将结合自己在 Apache Doris 社区支持 500 余家龙头企业完成数仓转型的经验，以及在 AI4Data 领域的实战落地专长，分享一套基于 Apache Doris 构建"零幻觉"AI 辅助决策平台的端到端解决方案。

本次演讲将破除业界对 AI4Data 的认知误区，聚焦企业真实的决策需求，向 Apache Doris 社区交付一套可复制、经验证的"AI + 数据引擎"融合实践框架，覆盖从技术架构设计到生产部署的完整生命周期：

  1. 零幻觉的技术底座：从"SQL 猜测"到"确定性生成"。我们创新的 NL2LF2SQL 双引擎架构，能够实现从自然语言到逻辑形式、再到 SQL 的逐步转换，彻底消除大模型数据查询中的幻觉问题，实现符合财务审计标准的 100% 数据准确率。依托 Apache Doris 的实时统一架构，AI 驱动的决策背后拥有毫秒级响应的算力支撑。
  2. 生产级落地实践：从原型到企业级部署。我们将分享在 Apache Doris 上构建 AI4Data 平台的最佳实践，包括如何直接复用 DWD 层明细数据、免去繁琐的中层构建，让企业以低成本构建智能决策平台，同时降低实施复杂度。
  3. 业务价值闭环：从"数据查询工具"到"战略决策顾问"。我们将演示如何将 AI 能力从基础数据查询延伸到归因分析、业务洞察生成、可执行决策支持的全价值链。借助我们"双脑协同"的数字员工模型，AI 不仅能精准查询数据，还能为企业提供可落地的决策建议，真正释放 AI4Data 的业务价值。

本次演讲将为 Apache Doris 社区用户带来一套兼顾技术深度与实用价值的 AI4Data 落地方案，助力企业在大模型时代构建值得信赖、高效且以价值为驱动的智能决策系统。

讲师:


奕嘉 苏：Yiwen Data 联合创始人、Apache Doris Committer、PowerData 发起人

Yiwen Data 联合创始人、Apache Doris Committer、Apache Doris 社区布道师、Doris-MCP 贡献者、PowerData 社区发起人，曾帮助数十家企业完成 Data Agent 与 ChatBI 项目的生产落地，致力于打造零幻觉的辅助决策平台，并协助 Apache Doris 社区中数百家企业完成实时数仓建设与优化演进。
