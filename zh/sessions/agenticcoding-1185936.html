用 AI Agent 构建 BanyanDB：Agentic Coding 工程实践

Youliang Huang

中文演讲 2026-08-09 13:30 GMT+8  (ROOM : 静明厅) #agenticcoding

本次分享将介绍 Agentic Coding 在 BanyanDB 研发中的实际应用。通过系统设计、代码实现、问题调试和测试等实践案例，展示工程师与 AI Agent 如何在软件开发的不同环节中协同工作。 分享不会局限于代码生成，而是将 Agentic Coding 视为一种工程协作流程：工程师负责定义目标、约束和架构原则，Agent 则协助完成探索、执行、验证和迭代。同时，分享还将总结相关实践经验，并讨论如何通过工程约束和验证机制，让 Agentic Coding 在复杂系统研发中更加可靠、可控。
