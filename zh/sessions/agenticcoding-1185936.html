借助 Agentic 工作流玩转 BanyanDB 追踪存储

Youliang Huang

中文演讲 2026-08-09 13:30 GMT+8  (ROOM : 静明厅) #agenticcoding

本次演讲探讨如何用 agentic coding（智能体编码）的方式来构建追踪（trace）存储能力，并深入剖析 BanyanDB 的 trace 存储引擎。我们不再遵循传统的、人工驱动的开发模式，而是展示 agentic coding 如何重塑工程工作流——从需求分析、系统设计，到实现与迭代优化。 通过把 agentic coding 的实践与 BanyanDB 专为 trace 设计的存储架构相结合，本次演讲展示了一种新的开发范式：借助智能体加速系统开发，并以 BanyanDB 的 trace 存储引擎作为面向 trace 原生（trace-native）工作负载系统的真实范例。
